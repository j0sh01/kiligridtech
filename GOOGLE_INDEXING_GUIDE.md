## üìã Step-by-Step Indexing Process

### Step 1: Google Search Console Setup (15 minutes)

#### A. Create/Access Google Search Console
1. Go to: https://search.google.com/search-console
2. Sign in with your Google account
3. Click **"Add Property"**
4. Select **"URL prefix"**
5. Enter: `https://kiligridtech.com`
6. Click **Continue**

#### B. Verify Ownership (Choose ONE method)

**Method 1: HTML File Upload (Easiest)**
1. Google will give you an HTML file to download (e.g., `google1234567890abcdef.html`)
2. Upload this file to your website's root directory
3. Make it accessible at: `https://kiligridtech.com/google1234567890abcdef.html`
4. Click **Verify** in Search Console

**Method 2: HTML Meta Tag**
1. Google will give you a meta tag like:
   ```html
   <meta name="google-site-verification" content="ABC123..." />
   ```
2. Add this to the `<head>` section of `index.html`
3. Deploy the updated site
4. Click **Verify** in Search Console

**Method 3: DNS Record (If you control DNS)**
1. Add TXT record to your domain's DNS
2. Wait for DNS propagation (can take hours)
3. Click **Verify**

---

### Step 2: Submit Sitemap (5 minutes)

Once verified:

1. In Google Search Console, go to **"Sitemaps"** (left sidebar)
2. Enter: `sitemap.xml`
3. Click **Submit**
4. Status should show: "Success" within a few minutes

**Your sitemap URL:** `https://kiligridtech.com/sitemap.xml`

---

### Step 3: Request Indexing for Key Pages (10 minutes)

1. In Search Console, go to **"URL Inspection"** (top search bar)
2. Enter each URL and click **"Request Indexing"**:
   - `https://kiligridtech.com/`
   - `https://kiligridtech.com/#/services`
   - `https://kiligridtech.com/#/solutions`
   - `https://kiligridtech.com/#/products`
   - `https://kiligridtech.com/#/about`
   - `https://kiligridtech.com/#/contact`

3. Google will say: "Indexing requested" (may take 1-7 days)

---

### Step 4: Bing Webmaster Tools (10 minutes)

Don't forget Bing! It's easier and faster:

1. Go to: https://www.bing.com/webmasters
2. Sign in with Microsoft account
3. Click **"Add a site"**
4. Enter: `https://kiligridtech.com`
5. Verify ownership (similar to Google)
6. Submit sitemap: `https://kiligridtech.com/sitemap.xml`

**Bonus:** Bing often indexes faster than Google!
